ABOUT

What is the vivarium monitor?

As a way to keep myself busy in the penultimate summer break of my degree, I decided to spend a couple of weeks designing, programming, and building a automatic vivarium from scratch. The vivarium monitor site is used as a centralised location to track the vivarium data.


What tools were used to create this?

An old Raspberry Pi B+ has been used as the brains of the project. The program used to monitor and control conditions is written in Python, and can be found here. Temperature and humidity readings are taken through a AM2302, processed using the Adafruits library, and uploaded to the cloud using the ThingSpeak API. The mains electricity is powered through a 'Pi-mote' and Energenie extension lead, as opposed to mains relay. Finally, the site has been created using HTML, CSS, and JavaScript, utilising the Bootstrap library.
